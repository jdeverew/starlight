This series walks through rolling out Microsoft Defender for Endpoint (MDE) end-to-end, from licensing prerequisites to day-two tuning. Each part builds on the previous one, but pages are self-contained enough to read on their own.

## In this series

1. [Part 1 — Introduction and prerequisites](/guides/defender-for-endpoint/01-introduction/)
2. [Part 2 — Onboarding Windows devices](/guides/defender-for-endpoint/02-onboarding/)
3. [Part 3 — Attack surface reduction rules](/guides/defender-for-endpoint/03-asr-rules/)

## Who this is for

- Security engineers rolling out MDE for the first time
- Administrators consolidating from a third-party EDR
- Analysts who want a reference for onboarding patterns

## How to use this series

Each page has a table of contents on the right. Use it to jump directly to the step you need. If you are reading straight through, a **Next** link at the bottom of each page points to the following part.
